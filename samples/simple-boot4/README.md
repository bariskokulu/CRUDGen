# Minimal usage — Spring Boot 4

Same **minimal** library profile as [simple-boot3](../simple-boot3/): one entity, `Read`+`Create`, security/lifecycle/OpenAPI/logging off.

## Stack

- Spring Boot **4.0.5**, MapStruct **1.6.3**
- `spring-boot-starter-webmvc`
- Separate source tree (not synced with Boot 3)

## Entity

`SimpleWidget` in `com.example.crudgen.simple` — mirror of Boot 3 minimal pattern.
